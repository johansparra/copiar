En IBM Cloud INTERFAZ WEB se tiene que crear el espacio de nombre
ahora hacerlo mismo en la consola

ibmcloud cr namespace-add <my_namespace>
ibmcloud cr namespace-add spacedatapower

Copiar Imagen ya existente de docker en este caso tenia la mia de datapower modificada

	PS C:\WINDOWS\system32> docker images
	REPOSITORY               TAG                 IMAGE ID            CREATED             SIZE
	johansparra/datapowerb   latest              d9f7bf2f69fa        16 hours ago        1.46GB
	ibmcom/datapower         latest              e9a591d8e681        2 months ago        1.45GB


docker tag <local_image> uk.icr.io/<my_namespace>/<my_repo>
docker tag d9f7bf2f69fa uk.icr.io/spacedatapower/datapowerbcloud

Se sube a IBM Cloud
docker push us.icr.io/<my_namespace>/<my_repo>
docker push uk.icr.io/spacedatapower/datapowerbcloud
