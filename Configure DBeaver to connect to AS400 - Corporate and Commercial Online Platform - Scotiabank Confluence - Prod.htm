    Configure DBeaver to connect to AS400

    Saltar al final de los metadatos
    • Creado por Edwin Bohorquez Buitrago, modificado por última vez en mar 02, 2021
    Ir al inicio de los metadatos

    • 2Installation

    Pre Configuration

    1. Check if Java is installed in the PC

      • In a CMD window, execute the next command:

        ?
        java -version

        The output will be something like this:

    2. If the command execution was not succesful, execute the next steps:
      1. Go to https://jdk.java.net/13/ and download the Windows / x64 zip
      2. Copy the downloaded file to D:\

      3. Open PowerShell and execute the next commands:

        ?
        1
        2
        3
        4
        D:
        Expand-Archive -Path .\openjdk-13_windows-x64_bin.zip -DestinationPath .
        cd .\jdk-13\
        .\bin\java.exe -version


    Installation

    1. Download DBeaver from https://dbeaver.io/download/ (Select the zip archive option)

    2. Copy the downloaded file to D:\ and extract the files

       Ampliar origen
      expand source?
      1
      2
      D:
      Expand-Archive -Path .\dbeaver-ce-6.2.1-win32.win32.x86_64.zip -DestinationPath .
    3. If Java was installed, open the file dbeaver.exe, otherwise follow the next steps:

      1. Create a file called "DBeaver..bat" and put the next content inside

        DBeaver.bat
        ?
        1
        2
        3
        4
        5
        6
        7
        @echo off
         
        set JAVA_HOME=D:\jdk-13
         
        SET PATH=%PATH%;%JAVA_HOME%\bin
         
        "D:\dbeaver\dbeaver.exe"
      2. Save the file and open with double click

    Configuration of the data source connection

    1. Open DBeaver
    2. In the dialog "Connect to database" click on "Cancel"
    3. Open DBeaver Preferences Window/Preferences (Ventana/Preferencias)
    4. In General/Network Connections (General/Conexiones de Red) change the active provider to Manual
    5. Select HTTP and click on Edit. Fill the fields with
      • Host: gateway.bns
      • Port: 8000
    6. Do the same with HTTPS

    7. Click on "Apply and Close"

    8. Open the Driver Manager "Database/Driver Manager" ("Base de Datos/Gestor de Driver")

    9. Find iSeries and click on Edit

    10. Click on "Download/Update"

    11. Click on Download

    12. Now in the "Edit Driver" dialog the color of the library should have changed to black instead of blue:

      Click on Accept

    13. Click on the button "Add Connection"

    14. Select "All" and find iSeries, select "DB2 iSeries/AS 400" and click on Next

    15. Fill the fields with this information:

      • Host: (See the table "AS400 IP's" below)

      • Port: The field must be empty

      • Database/Schema: Any library with read permissions (e.g. BDITSOL)

      • Username: The AS400 connection user

      • Password: Leave the field empty

      • Uncheck the field "Save password locally"

    16. Click on Test Connection, if the configuration was successful this dialog should appear:

      Click on Finish

    17. Select the connection and hit F3 to create a new SQL Sheet.
